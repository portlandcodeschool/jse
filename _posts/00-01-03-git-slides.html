Git & GitHub

Code History

Version control allows you to track the history of your code.

Collaboration

It also allows multiple people to work on the same project.

These systems do their best to avoid conflicts when collaborating, but they can't be perfect.

Why Git?

Free, open source, GitHub is awesome!

Other version control systems

  • Mercurial (hg), 2005
  • Subversion (svn), 2000
  • Concurrent Versions System (cvs), 1990
  • Perforce, 1995
  • Microsoft Visual Source Safe, 1994
Git was also 2005, GitHub was 2008. Mercurial, Subversion, and cvs are are open source. Mercurial is the only other distributed version control system (the rest are client server).
