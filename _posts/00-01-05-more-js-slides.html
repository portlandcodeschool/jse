JavaScript

The Parts We Skipped

Iterating


each(['water', 'coffee', 'tea', 'beer', 'wine'], function(beverage) {
  console.log('%s is delicious!', beverage);
});

We wrote our own each.

Iterating


['water', 'coffee', 'tea', 'beer', 'wine'].forEach(function(beverage) {
  console.log('%s is delicious!', beverage);
});

forEach and map are built in.

Node.js supports this, but not all older browsers do.

Iterating


var i = 0;
var array = ['water', 'coffee', 'tea', 'beer', 'wine'];
while (i < array.length) {
  var beverage = array[i];
  console.log('%s is delicious!', beverage);
  i++;
}

There's also these things called while loops.

Iterating


var array = ['water', 'coffee', 'tea', 'beer', 'wine'];
for (var i = 0; i < array.length; i++) {
  var beverage = array[i];
  console.log('%s is delicious!', beverage);
}

And for loops.

Things to Avoid

Bracket-less Conditions


if (beverage === 'water')
  healthy = true;
else if (beverage !== 'beer')
  healthy = true;

Only one statement is the body.

Type Coercion Equality Checks


if (beverage == 'water') {
  healthy = true;
}
else if (beverage != 'beer') {
  healthy = true;
}

These coerce the items being compared to the same type.
