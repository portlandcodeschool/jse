Modules

Problem

I want to reuse the each function in multiple projects I'm working on.

Problem

I wrote 1,000 lines of code all in one file, and now I can't find anything.

Problem

I want to do something that I'm pretty sure should be easy, but it's not built into JavaScript.

For instance, you may want to read a file or create a file.

Solution

Modules!

Modules

  • Code reuse & sharing
  • Separation of concerns (modularity)

Building a Home

  • Pour foundation
  • Construct frame
  • Add siding & windows
  • Install pipes and plumbing
  • Electrical
  • Insulation

Imagine building a home by yourself from start to finish. If instead you contract each part of the process out to other groups, you'll very likely get something finished faster, cheaper, and better than if you did it yourself.

Building a Website

Allow users to log in

  • Username/password
  • Twitter
  • Facebook
  • Google
  • OpenID
If comparing to the idea of building a house, you'd also probably need to create a way to store users on your own as well. So you'd be implementing your own database.

Creating Node Modules


/**
 * @file array-functions.js
 */

module.exports.each = function(array, fn) {
  // each function definition
};

module.exports is an object on which you can define additional properties.

Using Node Modules


/**
 * @file app.js
 */

var arrayFunctions = require('./array-functions');
var fruits = ['kiwi', 'strawberry', 'banana'];

arrayFunctions.each(fruits, function(fruit) {
  addFruitToSmoothie(fruit);
});

require allows you to access the module.exports object defined in the other file.

Documenting Modules

Specifically formed comments can document code.


/**
 * Enumerate an array
 * @function
 * @param {array} array - The array to enumerate
 * @param {function} fn - The function to call for each item
          in the array. It only takes one argument.
 */
module.exports.each = function(array, fn) {
};
Comments usually only help us understand code, but this special syntax is used by JSDoc to create documentation.
