map(array, fn) → {function}

Create a new array by applying fn to each element in array and using each result as the items of the new array.

Examples
// create an array of squares
map([1,2,3], function(n) {
  return n * n;
});
//=> [1, 4, 9]
// make all words plural (simple logic)
var pluralize = function(word) {
  return word + 's';
}
map(['dog','cat'], pluralize);
//=> ['dogs', 'cats']
